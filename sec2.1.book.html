Section 2.1 Linear First Order Equations
2.1 LINEAR FIRST ORDER EQUATIONS
Linear First Order Equations
A first order differential equation is said to be linear if it can be written in standard form as
\begin {equation} \label {eq:2.1.1} \frac{dy}{dx} + p(x)y = f(x) \end {equation} Any other first order differential equation that can’t be written in this form is nonlinear.
We say that a first order linear equation (2.1.1) is homogeneous if \(f(x) \equiv 0\): \[ \frac{dy}{dx} + p(x)y = 0\] Otherwise for any \( f(x) \not\equiv 0 \) (2.1.1) is nonhomogeneous.
Since \(y(x) \equiv 0\) is obviously a solution of the homgeneous equation we call it the trivial solution. \[ \frac{d}{dx}\Big(0\Big)+p(x)\cdot\Big(0\Big) = 0 \quad \checkmark \] Any other solution is nontrivial.
Rewrite the linear first order differential equations in standard form, idenitfy \( p(x) \) and \( f(x) \), and state whether the equation is homogeneous or nonhomogeneous
\begin {eqnarray*} x^2 \frac{dy}{dx} + 3y& = &x^2 \\ x \frac{dy}{dx} -8x^2y& = &\sin x \\ x \frac{dy}{dx} + (\ln x)y& = &0 \\ \frac{dy}{dx} & = &x^2y - 2 \end {eqnarray*}
Solution
Rearranging the equations into standard form (2.1.1), we can readily identify \( p(x) \) and \( f(x)\): \begin{align*} \frac{dy}{dx} + \overset{p(x)=\frac{3}{x^2}}{\overbrace{\frac{3}{x^2} }}y = & \overset{f(x)=1}{\overbrace{1}} \\ \\ \frac{dy}{dx} - \overset{p(x)=-8x}{\overbrace{8x }}y = & \overset{f(x)=\frac{\sin x}{x}}{\overbrace{\frac{\sin x}{x}}} \\ \\ \frac{dy}{dx} + \overset{p(x)=\frac{1}{x}\ln x}{\overbrace{\frac{1}{x}\ln x }}y = & \overset{f(x)=0}{\overbrace{0}} \\ \\ \frac{dy}{dx} - \overset{p(x)=-x^2}{\overbrace{x^2 }}y = & \overset{f(x)=-2}{\overbrace{-2}} \\ \\ \end{align*} The third equation \( \displaystyle \frac{dy}{dx} + \frac{1}{x}\ln x = 0 \) is homogeneous since \(f(x) \equiv 0\) and all others are nonhomogeneous
Explain how the following first order equations cannot be written in the linear standard form (2.1.1) and thus are nonlinear \begin{align*} x\frac{dy}{dx} + 3y^2 = & 2x \\ y\frac{dy}{dx} = & 3 \\ \frac{dy}{dx} + xe^y = & 12 \\ \end{align*}
Solution The linear standard form is: \( \displaystyle \frac{dy}{dx} + p(x)y = f(x) \). The following equations cannot be written in this form. \begin{align*} x\frac{dy}{dx} + 3y^2 = 2x & \quad \text{is nonlinear as $y$ is squared} \\ y\frac{dy}{dx} = 3 & \quad \text{is nonlinear as there is a product of $y$ and its derivative $\frac{dy}{dx}$} \\ \frac{dy}{dx} + xe^y = 12 & \quad \text{is nonlinear as $e^y$ is a non-linear function of $y$} \\ \end{align*}
General Solution of Linear First Order Equations
Consider the simple linear first order equation: \begin {equation} \label {eq:2.1.2} \frac{dy}{dx} = {1\over x^2} \end {equation} We notice it is integrable since we can immediately find the antiderivative using integration: \begin {equation} \label {eq:2.1.3} y = -{1\over x} + c \end {equation}
Show the work to find the antiderivative of (2.1.2)
We call the arbitrary constant of integration \(c\) a parameter and we say that (2.1.3) defines a one–parameter family of functions. For each real number \(c\), the function defined by (2.1.3) is a solution of (2.1.2) defined on the intervals of validity \((-\infty ,0)\) or \((0,\infty) \) and when every solution of (2.1.2) on either of these intervals is of the form of (2.1.3) for some choice of \(c\), we say that (2.1.3) is the general solution of (2.1.2)
Although (2.1.2) was simple and integrable, the same vocabulary summary applies for all 1st order linear differential equations: \begin {equation} \label {eq:2.1.4} \frac{dy}{dx} + p(x)y = f(x) \end {equation} That is, if \(p(x)\) and \(f(x)\) are continuous on some open interval \((a,b)\) then there’s a unique formula \(y = y(x,c)\) analogous to (2.1.3) that involves \(x\) and a parameter \(c\) that has the these properties:
  • For each value of \(c\), the resulting function \( y=y(x) \) is a solution of (2.1.4) on an interval of validity \((a,b)\)
  • A solution of (2.1.4) can be obtained from the equation representing the one-parameter family by choosing \( c \) appropriately
  • We call \( y = y(x,c) \) the general solution and for a particular value of \( c \) then \( y=y(x) \) is a particular solution
So far we have only considered first order linear differential equations in the form: \begin{equation*} \frac{dy}{dx} + p(x)y = f(x) \end{equation*} However, first order linear differential equations could also be of the form: \begin {equation} \label {eq:2.1.5} P_0(x) \frac{dy}{dx} + P_1(x)y = F(x) \end{equation} which has a general solution provided \(P_0(x)\), \(P_1(x)\), and \(F(x)\) are all continuous on an open interval \((a,b)\) on which and \(P_0(x) \) has no zeros, since in this case we can rewrite it in the form of (2.1.4) by dividing by \( P_0(x) \): \begin {equation*} \frac{dy}{dx} + \underset{p(x)}{\underbrace{\frac{P_1(x)}{P_0(x)}}}y = \underset{f(x)}{\underbrace{\frac{F(x)}{P_0(x)}}} \end{equation*} However, take note of the fact that the intervals of validity might change when dividing by \( P_0(x) \) if \( P_0(x) \) has any zero on an interval we are considering (Exercise 44 for an example of this), and whenever we find a solution, we should include every open interval of validity for which the solution is defined.
Homogeneous Linear First Order Equations
We begin with the problem of finding the general solution of a homogeneous linear first order equation. The next example recalls a familiar result from calculus.
  1. Let \(a\) be a constant. Find the general solution: \begin {equation} \label{eq:2.1.6} \frac{dy}{dx} - ay = 0 \end {equation}
  2. Solve the initial value problem \[ \frac{dy}{dx} -ay = 0 \quad y(x_0) = y_0\]
Solution
  1. First notice (2.1.6) has a trivial solution \( y(x) \equiv 0 \) which we can verify by substitution: \begin{align*} \frac{dy}{dx} - ay = & 0 \\ \frac{d}{dx} \Big( 0 \Big) - a \Big( 0 \Big) = & 0 \quad \checkmark \\ \end{align*}
    Now we seek nontrivial solution where \( y \ne 0 \) by rearranging the equation: \begin{align*} \frac{dy}{dx} - ay = & 0 \\ \frac{dy}{dx} = & ay \\ \frac{1}{y}\frac{dy}{dx} = & a \\ \int \Bigg( \frac{1}{y}\frac{dy}{dx} \Bigg) dx = & \int \Big( a \Big) dx \\ \int \frac{1}{y} dy = & a\int dx \\ \end{align*}
    Notice how the equation has the variables separated so that the left side has only \( y \) and the right side has only \( x \). This strategy is called separation of variables. Now find the antiderivative on both sides. Technically each side has an arbitrary constant but we combine them into one constant \( k \). \begin{align*} \int \frac{1}{y} dy = & a\int dx \\ \ln \left| y \right| = & ax + k \end{align*}
    Solving for \( y \) requires exponeniating both sides resulting in \( e^l \) in the expression, which is just another arbitrary constant so we can simplify it to just \( K \) \begin{align*} \ln \left| y \right| = & ax + k \\ e^{\ln \left| y \right|} = & e^{ax + k} \\ \left| y \right| = & e^{ax} \underset{K}{\underbrace{e^{k}}} \\ \left| y \right| = & Ke^{ax} \\ y = Ke^{ax} \quad & \text{or} \quad y = -Ke^{ax} \end{align*} Since the function \( e^{ax} \) never equals zero so it is either always positive or always negative and with \( K \) and\( -K \) representing arbitrary constants, we can further simplify \begin {equation} \label {eq:2.1.7} y(x) = ce^{ax} \end{equation}
    This shows that every nontrivial solution of (2.1.6) is of the form \(y = ce^{ax}\) for some nonzero constant \(c\). Since the trivial solution is also represented by this formula if we set \(c = 0\), then all solutions of (2.1.6), both trivial and nontrivial, are of the form of (2.1.7).
    Additionally, we can check our answer simplify by differentiating and rewriting it in terms of (2.1.6): \begin{align*} y = ce^{ax} \Longrightarrow \frac{dy}{dx} = & \frac{d}{dx} \Big(ce^{ax} \Big) \\ \frac{dy}{dx} = & ce^{ax} \underset{a}{\underbrace{\frac{d}{dx}\left({ax}\right)}} \\ \frac{dy}{dx} = & a \cdot \underset{y}{\underbrace{ce^{ax}}} \\ \frac{dy}{dx} = & a y \Longrightarrow \frac{dy}{dx} -ay = 0 \quad \checkmark \end{align*}
  2. We already have the general solution (2.1.7) and we evaluate it at our initial condition \(y(x_0) = y_0\) to solve for the constant \( c \): \begin{align*} y(x) = & ce^{ax} \\ \underset{y_0}{\underbrace{y(x_0)}} = & ce^{ax_0} \\ y_0 = & ce^{ax_0} \\ c = & \frac{y_0}{e^{ax_0}} = y_0 e^{-ax_0} \\ \end{align*} Substituting \( c = y_0 e^{-ax_0} \) yields the particular solution of our IVP: \begin{align*} y(x) = ce^{ax} \Longrightarrow y(x) = & y_0 e^{-ax_0} e^{ax} \\ y(x) = & y_0 e^{-ax_0+ax} \\ y(x) = & y_0 e^{a\left(x-x_0\right)} \\ \end{align*}

  1. Find the general solution of \begin {equation} \label{eq:2.1.8} x \frac{dy}{dx} + y = 0 \end {equation}
  2. Solve the initial value problem \begin {equation} \label {eq:2.1.9} x \frac{dy}{dx} + y = 0\quad y(1) = 3 \end {equation}
Solution
  1. First notice (2.1.8) has a trivial solution \( y(x) \equiv 0 \) which we can verify by substitution: \begin{align*} x\frac{dy}{dx} + y = & 0 \\ x\frac{d}{dx} \Big( 0 \Big) + \Big( 0 \Big) = & 0 \quad \checkmark \\ \end{align*}
    Next divide (2.1.8) by \( x \) to rewrite in standard form (2.1.1) \begin {equation} \label {eq:2.1.10} \frac{dy}{dx} + \frac{1}{x} y = 0 \end {equation} Now \( x \ne 0 \) since we divided by it, so that limits our intervals of validity to either \((-\infty ,0)\) or \((0,\infty )\). Since we have already accounted for the trivial solution, we now seek nontrivial solution where \( y \ne 0 \) by rearranging the equation using separation of variables: \begin{align*} \frac{dy}{dx} + \frac{1}{x}y = & 0 \nonumber \\ \frac{dy}{dx} = & -\frac{1}{x}y \nonumber \\ \frac{1}{y}\frac{dy}{dx} = & -\frac{1}{x} \nonumber \\ \int \Bigg( \frac{1}{y}\frac{dy}{dx} \Bigg) dx = & \int \Bigg( -\frac{1}{x} \Bigg) dx \nonumber \\ \int \frac{1}{y} dy = & - \int \frac{1}{x} dx \nonumber \\ \end{align*} Finding the antiderivatives and simplifying, \begin{align*} \ln \left| y \right| = & - \ln \left| x \right| + k \nonumber \\ \ln \left| y \right| + \ln \left| x \right| = & k \nonumber \\ \ln \Big( \left| yx \right| \Big) = & k \nonumber \\ e^{\ln \big( \left| yx \right| \big)} = & \underset{K}{\underbrace{e^{k}}} \nonumber \\ \left| yx \right| = & K \nonumber \\ yx = K \quad &\text{or} \quad yx = -K \\ \end{align*}
    Since \( x \) is never zero on our intervals \((-\infty ,0)\) and \((0,\infty )\), the expression is either always positive or always negative and with \( K \) and\( -K \) representing arbitrary constants, we can further simplify \begin {equation} \label {eq:2.1.11} y(x) = \frac{c}{x} \end{equation} This shows that every nontrivial solution of (2.1.10) is of the form (2.1.11) for some nonzero constant. And even though we assumed \( y \ne 0 \) (i.e. was not the trivial solution) in order to find this solution, it turns out the trivial solution is still represented by the formula if we set \(c=0\), thus all solutions of (2.1.10), both trivial and nontrivial, are of the form of (2.1.11), hence it is our general solution. Additionally, we can check our answer simplify by differentiating \begin{align*} \frac{dy}{dx} = & \frac{d}{dx} \Bigg( \frac{c}{x} \Bigg) = c\frac{d}{dx} \Big( x^{-1} \Big) = -cx^{-2} = -\frac{c}{x^2} \\ \end{align*} And substituting the general solution (2.1.11) and its derivative into (2.1.10). \begin{align*} \frac{dy}{dx} + \frac{1}{x} y = & 0 \\ -\frac{c}{x^2} + \frac{1}{x} \Bigg( \frac{c}{x} \Bigg) \overset{?}{=} & 0 \\ -\frac{c}{x^2} + \frac{c}{x^2} = & 0 \quad \checkmark \end{align*}
  2. We already have the general solution (2.1.11) and we evaluate it at our initial condition \(y(1) = 3\) to solve for the constant \( c \): \begin{align*} y(x) = & \frac{c}{x} \\ \underset{3}{\underbrace{y(1)}} = & \frac{c}{1} \\ c = & 3 \\ \end{align*} Substituting \( c = 3 \) yields the particular solution of our IVP (2.1.9): \begin{align*} y(x) = & \frac{c}{x} \Longrightarrow y(x) = \frac{3}{x} \\ \end{align*} Since \( x = 1 \) then the interval of validity for this solution is \((0,\infty )\)

Nonhomogeneous Linear First Order Equations
In Examples 2.1.3 and 2.1.4, we rewrote homogeneous linear equations so that one side depended on only \( y \) and its derivative and the other side only depended on \(x\) and then found the antiderivative on both sides. This method is called separation of variables and can be used to solve all homogeneous linear equations but now we would like to solve nonhomogeneous equations: \begin {equation} \label{eq:2.1.15} \frac{dy}{dx} + p(x)y = f(x) \end {equation} To solve nonhomogeneous equations (2.1.15), we must first solve the corresponding homogeneous equation, which we call the complementary equation: \[ \frac{dy}{dx} + p(x)y = 0 \] We then use a method called variation of parameters to seek solutions of the form \( y = u y_1 \) where \( y_1 = y_1 (x) \) is a nontrivial solution of the complementary equation and \( u = u(x) \) is some function of \( x \) to be determined. Since \( y_1 \) is a solution of the complementary equation, then we know and will use the fact that \begin{equation} \label{eq:2.1.16} \frac{dy_1}{dx} + p(x)y_1 = 0 \end{equation}
To implement variation of parameters, we will differentiate \( y = uy_1 \) and then substitutute back into our nonhomogeneous equations (2.1.15) to solve. We differente using the product rule to get an expression for the derivative in terms of \( u \) and \( y_1 \): \begin{align*} \frac{dy}{dx} = & \frac{d}{dx} \Big( u \cdot y_1 \Big) = y_1\frac{du}{dx} + u\frac{dy_1}{dx} \end{align*}
Substituting \( y = uy_1 \) and its derivative into (2.1.15) then simplifying the equation it transforms into an integrable equation (2.1.17): \begin{align} \label{eq:2.1.17} \frac{dy}{dx} + p(x)y = & f(x) \nonumber \\ \overset{\frac{dy}{dx}}{\overbrace{y_1\frac{du}{dx} + u\frac{dy_1}{dx}}} + p(x) \overset{y}{\overbrace{u \cdot y_1}} = & f(x) \nonumber \\ y_1\frac{du}{dx} + u \underset{0}{ \underbrace{ \left( \frac{dy_1}{dx} + p(x)y_1 \right)} } = & f(x) \nonumber \\ y_1\frac{du}{dx} + u \cdot 0 = & f(x) \nonumber \\ y_1\frac{du}{dx} = & f(x) \nonumber \\ \frac{du}{dx} = & \frac{f(x)}{y_1(x)} \end{align}
Now we simply find the antiderivative to find \( u \), which introduces a arbitrary constant of integration. Finally, multiply the result \( u \) by \(y_1\) to get \( y = uy_1 \): \begin{align*} \frac{du}{dx} = & \frac{f(x)}{y_1(x)} \\ u = & c + \int \frac{f(x)}{y_1(x)} dx \\ y = & y_1 \left( c + \int \frac{f(x)}{y_1(x)} dx \right) \\ \end{align*}
Find the general solution of \begin {equation} \label {eq:2.1.18} \frac{dy}{dx} + 2y = x^3e^{-2x}\end {equation}
Solution
To solve the nonhomogeneous (2.1.18), always first solve the complementary homogeneous equation: \begin{align*} \frac{dy}{dx} + 2y = & 0 \end{align*}
Solve \( \frac{dy}{dx} + 2y = 0 \) using separation of variables as in Examples 2.1.3 and 2.1.4
The general solution for the complementary equation is \( y = ce^{-2x} \) and when using variation of parameters we select any simple particular solution so we choose \( c = 1 \) and use \( y_1 = e^{-2x} \) to seek solutions of (2.1.18) of the form: \[ y = u \cdot y_1 = u \cdot e^{-2x} \]
Differentiating using the product rule gives us: \begin{align*} \frac{dy}{dx} = & \left( \frac{du}{dx} - 2 u \right) e^{-2x} \\ \end{align*}
Differentiate \( u \cdot e^{-2x} \) using the product and chain rules
Now substitute into (2.1.18) and by simplifying the equation it transforms into an integrable equation (2.1.19): \begin{align}\label{eq:2.1.19} \frac{dy}{dx} + 2y = & x^3 e^{-2x} \nonumber \\ \overset{\frac{dy}{dx}}{\overbrace{ \left( \frac{du}{dx} - 2 u \right) e^{-2x} }} + 2\overset{y}{\overbrace{ue^{-2x}}} = & x^3 e^{-2x} \nonumber \\ \frac{du}{dx}e^{-2x} \underset{0}{\underbrace{- 2 u e^{-2x} + 2ue^{-2x}}} = & x^3 e^{-2x} \nonumber \\ \frac{du}{dx}e^{-2x} = & x^3 e^{-2x} \nonumber \\ \frac{du}{dx} = & x^3 \end{align} Now we simply find the antiderivative to find \( u \): \begin{align*} \int du = & \int x^3 dx \Longrightarrow u = c + \frac{1}{4} x^4 \\ \end{align*} Then multiply the result by \(y_1 = e^{-2x} \) to get the general solution of (2.1.18): \begin{align*} y = ue^{-2x} = & \left( c + \frac{1}{4} x^4 \right)e^{-2x} \\ \end{align*}

Variation of Parameters for First Order Linear Equations \begin {equation} \label {eq:2.1.26} \frac{dy}{dx} + p(x)y = f(x) \end {equation}
Summary of the Steps To Solve (2.1.26)
  1. Solve the complementary homogeneous equation \( \frac{dy}{dx} + p(x)y = 0 \) and select a simple particular solution as \( y_1(x) \)
  2. Express \( y \) as a product \( y = u \cdot y_1 \) and calculate its derivative \( \frac{d}{dx} \Big( u \cdot y_1 \Big) \) using product, quotient, and/or chain rules
  3. Substitute \( uy_1 \) and \( \frac{d}{dx} \Big( u \cdot y_1 \Big) \) into the nonhomogeneous (2.1.26) and simplify to transform into an integrable equation
  4. Find the antiderivative \( u \) including the constant of integration, and multiply result by \( y_1 \) to get \( y = uy_1 \)
Solutions in Integral Form
Sometimes the integrals that arise in solving a linear first order equation can’t be evaluated in terms of elementary functions. In this case the solution must be left in terms of an integral.
  1. Find the general solution of \begin {equation} \label {eq:2.1.27} \frac{dy}{dx} -2xy = 1 \end {equation}
  2. Solve the initial value problem \begin {equation} \label {eq:2.1.28} \frac{dy}{dx} - 2xy = 1 \quad y(0) = y_0 \end {equation}
Solution
  1. To solve (2.1.27), first use separation of variables to solve the complementary homogeneous equation: \begin{align*} \frac{dy}{dx} - 2xy = & 0 \end{align*}
    Show all the work to solve \( \frac{dy}{dx} - 2xy = 0 \) as in Examples 2.1.3 and 2.1.4
    Our general solution for the complementary equation is \( y = ce^{x^2} \) and to use variation of parameters we select a simple particular solution of the complementary equation so we choose \( c = 1 \) and use \( y_1 = e^{x^2} \) and seek solutions of the form \( y = u \cdot y_1 = u \cdot e^{x^2} \)
    Differentiating \( y = u \cdot e^{x^2} \) using the product rule gives us: \begin{align*} \frac{dy}{dx} = & \left( \frac{du}{dx} + 2 x u \right) e^{x^2} \\ \end{align*}
    Show all the work to calculate \( \frac{d}{dx} \Big( u \cdot e^{x^2} \Big) \) using the product and chain rules
    Now we substitute into (2.1.27) and by simplifying the equation transforms into an integrable equation: \begin{align*} \frac{dy}{dx} - 2 x y = & 1 \nonumber \\ \overset{\frac{dy}{dx}}{\overbrace{ \left( \frac{du}{dx} + 2 x u \right) e^{x^2} }} - 2x\overset{y}{\overbrace{ue^{x^2}}} = & 1 \nonumber \\ \frac{du}{dx}e^{x^2} - \underset{0}{\underbrace{ 2 x u e^{x^2} + 2xue^{x^2}}} = & 1 \nonumber \\ \frac{du}{dx}e^{x^2} = & 1 \nonumber \\ \frac{du}{dx} = & e^{-x^2} \end{align*} Normally we integrate to find \( u \) but \( e^{-x^2} \) can not be integrated by any of the methods we know from calculus, and in fact, cannot be expressed in terms of elementary functions. No problem, using Fundamental Theorem of Calculus we can always write an antiderivative as a definite integral \begin{align*} \int du = & \int e^{-x^2} dx \Longrightarrow u = c + \int_{x_0}^x e^{-t^2} dt \end{align*} where \( x_0 \) is the initial \(x-\)value. Of course this always introduces a constant of integration so be sure to include that.
    Verify \( \frac{du}{dx} = e^{-x^2} \) by differentiating \( u = c + \int_{x_0}^x e^{-t^2} dt\) using the Fundamental Theorem of Calculus
    Then multiply the result by \(y_1 = e^{x^2} \) to get the general solution \( y \) of (2.1.27): \begin{align} \label {eq:2.1.29} y = ue^{x^2} = & e^{x^2}\left( c + \int_{0}^x e^{-t^2} dt \right) \\ \end{align}
    Verify \( y = e^{x^2}\left( c + \int_{0}^x e^{-t^2} dt \right) \) is a solution of (2.1.27) by differentiating and substituting
  2. We already have the general solution (2.1.29) and we evaluate it at our initial condition \( y(0) = y_0 \) to solve for the constant \( c \): \begin{align*} y(x) = & e^{x^2}\left( c + \int_{0}^x e^{-t^2} dt \right) \\ \underset{y_0}{\underbrace{y(0)}} = & e^{0^2}\left( c + \int_{0}^0 e^{-t^2} dt \right) \\ y_0 = & 1 \left( c + 0 \right) \\ c = & y_0 \end{align*} Substituting \( c = y_0 \) yields the particular solution of our IVP: \begin{align} \label {eq:2.1.30} y(x) = e^{x^2}\left( c + \int_{0}^x e^{-t^2} dt \right) \Longrightarrow y(x) = & e^{x^2}\left( y_0 + \int_{0}^x e^{-t^2} dt \right) \end{align}
Figure 2.1.5 shows graphs of (2.1.30) for several values of \(y_0\).
A note about solutions in the form of an integral. Comparing to our experience in calculus, a general solution in the form of an integral (2.1.30) may seem incomplete, almost like we still have more steps to do; while this may be true in calculus class, it is important to realize that a solution in the form or a definite integral is perfectly valid and can be easily analyzed using numerical methods. For instance, remember that for any specific value of \( x \), this definite integral will evaluate to a number, so we could always use a calculator to make a table of \( \big( x, y(x) \big) \) pairs and simply plot the points of the solution. While we won’t be studying numerical methods in this class, hopefully you could imagine someone doing so and can be okay with sometimes having answers in the form of an integral.

Existence and Uniqueness
Theorem 2.1.2 Suppose \(p(x)\) and \(f(x)\) are continuous on an open interval \((a,b),\) and let \(y_1 = y_1(x) \) be any nontrivial solution of the complementary equation \( \displaystyle \frac{dy}{dx} + p(x)y = 0 \) on \((a,b)\). Then:
  1. The general solution of the nonhomogeneous equation (2.1.31) on \((a,b)\) is given by (2.1.32) \begin {equation} \label {eq:2.1.31} \frac{dy}{dx} + p(x)y = f(x) \end {equation} \begin {equation} \label {eq:2.1.32} y = y_1 \left (c + \int \frac{f(x)}{y_1(x)} dx \right ) \end {equation}
  2. If \(x_0\) is an arbitrary point in \((a,b)\) and \(y_0\) is an arbitrary real number, then the initial value problem (2.1.33) has the unique solution (2.1.34) on \((a,b)\) \begin {equation} \label {eq:2.1.33} \frac{dy}{dx} + p(x)y = f(x) \quad y(x_0) = y_0 \end {equation} \begin {equation} \label {eq:2.1.34} y = y_1 \left( {y_0\over y_1(x_0)} + \int^x_{x_0} {f(t)\over{y_1(t)}} dt \right) \end {equation}
Use the Fundamental Theorem of Calculus to differentiate (2.1.32) and use the solution to the complementary equation \( y_1 \) to substitute and show that (2.1.32) is a solution of (2.1.31)
Differentiate \( u = \frac{y}{y_1} \) using the quotient rule and use the solution to the complementary equation \( y_1 \) and (2.1.31) to substitute to show that \( \frac{du}{dx} = \frac{f(x)}{y_1(x)} \)
Evaluate (2.1.34) at \( x = x_0 \) to verify \( y( x_0 ) = y_0 \). Next evaluate (2.1.32) at \( x = x_0 \) and solve for \( c \) to show that \( c = \frac{y(x_0)}{y_1(x_0)} \)
