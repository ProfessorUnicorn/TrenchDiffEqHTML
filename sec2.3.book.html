Section 2.3 Existence and Uniqueness of Solutions of Nonlinear Equations
2.3   EXISTENCE AND UNIQUENESS OF SOLUTIONS OF NONLINEAR EQUATIONS
Open Rectangle
Although there are methods for solving some nonlinear equations, it’s impossible to find useful formulas for the solutions of most. Whether we’re looking for exact solutions or numerical approximations, it’s useful to know conditions that imply the existence and uniqueness of solutions of initial value problems for nonlinear equations. In this section we state such a condition and illustrate it with examples.
Some terminology: an open rectangle \(R\) (Shown in Figure 2.3.1) is a set of points \((x,y)\) such that \[ a<x<b \quad \mbox {and} \quad c<y<d \] We’ll denote this set by \(R: \{ a < x < b, c < y < d \}\). "Open" means that the boundary rectangle isn’t included in the set \(R\)
Existence and Uniqueness Theorem
The next theorem gives sufficient conditions for existence and uniqueness of solutions of IVPs for first order nonlinear differential equations.
Theorem 2.3.1
  1. If \( f(x,y) \) is continuous on an open rectangle \( R: \{ a < x < b, c < y < d \} \) that contains \( (x_0,y_0),\) then the IVP has at least one solution on some open subinterval of \( (a,b) \) that contains \( x_0 \) \begin {equation} \label {eq:2.3.1} \frac{dy}{dx}=f(x,y) \quad\quad y(x_0)=y_0 \ \end {equation}
  2. If both \(f(x,y)\) the function and \(f_y\) the partial derivative with respect to \(y\) are continuous on \(R\) then (2.3.1) has a unique solution on some open subinterval of \((a,b)\) that contains \(x_0\)
It’s important to understand exactly what Theorem  2.3.1 says.
  • (a) is an existence theorem. It guarantees that a solution exists on some open interval that contains \(x_0\), but provides no information on how to find the solution, or to determine the open interval on which it exists. Moreover, (a) provides no information on the number of solutions that (2.3.1) may have. It leaves open the possibility that (2.3.1) may have two or more solutions that differ for values of \(x\) arbitrarily close to \(x_0\). We will see in Example 2.3.6 that this can happen.
  • (b) is a uniqueness theorem. It guarantees that (2.3.1) has a unique solution on some open interval (a,b) that contains \(x_0\). However, if \((a,b)\ne (-\infty ,\infty )\), (2.3.1) may have more than one solution on a larger interval that contains \((a,b)\). For example, it may happen that \(b<\infty \) and all solutions have the same values on \((a,b)\), but two solutions \(y_1\) and \(y_2\) are defined on some interval \((a,b_1)\) with \(b_1>b\), and have different values for \(b<x<b_1\); thus, the graphs of the \(y_1\) and \(y_2\) “branch off” in different directions at \(x=b\). (See Example 2.3.7 and Figure 2.3.3). In this case, continuity implies that \(y_1(b)=y_2(b)\) (call their common value \(\overline y\)), and \(y_1\) and \(y_2\) are both solutions of the initial value problem \begin {equation} \label {eq:2.3.2} \frac{dy}{dx}=f(x,y),\quad y(b)=\overline y \end {equation} that differ on every open interval that contains \(b\). Therefore \(f\) or \(f_y\) must have a discontinuity at some point in each open rectangle that contains \((b,\overline y)\), since if this were not so, (2.3.2) would have a unique solution on some open interval that contains \(b\). We leave it to you to give a similar analysis of the case where \(a>-\infty \).
Existence and Uniqueness Examples
Determine whether the IVP has a unique solution using Theorem 2.3.1 \begin {equation} \label {eq:2.3.3} \frac{dy}{dx}={x^2-y^2 \over 1+x^2+y^2} \quad\quad y(x_0)=y_0 \end {equation}
Solution
Identify \( f(x,y) \) using (2.3.1) and check whether it is continuous: \begin{equation*} \frac{dy}{dx} = \overset{f(x,y)}{\overbrace{\frac{x^2-y^2}{1+x^2+y^2}}} \end{equation*} Since the denominator \( 1+x^2+y^2 \) is the sum of all positive quantities, \( f(x,y) = \displaystyle \frac{x^2-y^2}{1+x^2+y^2} \) is continuous for all \((x,y)\) so Theorem 2.3.1 (a) indicates that there exists at least one solution to the IVP.
So we know there exists a solution, now we will find out if it is unique by finding the partial derivative with respect to \( y \) \begin{align*} f_y(x,y) = & \frac{\partial}{\partial y}\left(\frac{x^2-y^2}{1+x^2+y^2}\right) \\ = & \frac{\left(1+x^2+y^2\right) \frac{\partial}{\partial y} \left(x^2-y^2\right) - \left(x^2-y^2\right) \frac{\partial}{\partial y} \left(1+x^2+y^2\right)}{\left(1+x^2+y^2\right)^2} \\ = & \frac{\left(1+x^2+y^2\right) \left(-2y\right) - \left(x^2-y^2\right) \left(2y \right)}{\left(1+x^2+y^2\right)^2} \\ = & \frac{ -2y \Big[ \left(1+x^2+y^2\right) + \left(x^2-y^2\right) \Big] }{\left(1+x^2+y^2\right)^2} \\ = & -\frac{2y\left(1+2x^2\right)}{\left(1+x^2+y^2\right)^2} \\ \end{align*} Since the denominator \( 1+x^2+y^2 \) is the sum of all positive quantities, \( f_x(x,y) = -\displaystyle \frac{2y\left(1+2x^2\right)}{\left(1+x^2+y^2\right)^2} \) is continuous for all \((x,y).\) Since both \( f(x,y) \) and \( f_x(x,y) \) are continuous then Theorem 2.3.1 (b) indicates that there exists a unique solution for (2.3.3)
Determine whether the IVP has a unique solution using Theorem 2.3.1 \begin {equation} \label {eq:2.3.4} \frac{dy}{dx} = \frac{x^2-y^2}{x^2+y^2} \end {equation}
Solution
Identify \( f(x,y) \) using (2.3.1) and check whether it is continuous: \begin{equation*} \frac{dy}{dx} = \overset{f(x,y)}{\overbrace{\frac{x^2-y^2}{x^2+y^2}}} \end{equation*} Aside from \((x,y) = (0,0),\) the denominator \( x^2+y^2 \) is the sum of all positive quantities, so \( f(x,y) = \displaystyle \frac{x^2-y^2}{x^2+y^2} \) is continuous for all \((x,y) \ne (0,0) \) so Theorem 2.3.1 (a) indicates that there exists at least one solution to the IVP.
So we know there exists a solution, now we will find out if it is unique by finding the partial derivative with respect to \( y \) \begin{align*} f_y(x,y) = & \frac{\partial}{\partial y}\left(\frac{x^2-y^2}{x^2+y^2}\right) \\ = & \frac{\left(x^2+y^2\right) \frac{\partial}{\partial y} \left(x^2-y^2\right) - \left(x^2-y^2\right) \frac{\partial}{\partial y} \left(x^2+y^2\right)}{\left(x^2+y^2\right)^2} \\ = & \frac{\left(x^2+y^2\right) \left(-2y\right) - \left(x^2-y^2\right) \left(2y \right)}{\left(x^2+y^2\right)^2} \\ = & \frac{ -2y \Big[ \left(x^2+y^2\right) + \left(x^2-y^2\right) \Big] }{\left(x^2+y^2\right)^2} \\ = & -\frac{2y\left(2x^2\right)}{\left(x^2+y^2\right)^2} \\ = & -\frac{4 x^2 y }{\left(x^2+y^2\right)^2} \\ \end{align*} Again aside from \((x,y) = (0,0),\) the denominator \( x^2+y^2 \) is the sum of all positive quantities, \( f_x(x,y) = -\displaystyle \frac{4 x^2 y }{\left(x^2+y^2\right)^2} \) is continuous for all \((x,y) \ne (0,0). \) Since both \( f(x,y) \) and \( f_x(x,y) \) are continuous for all \((x,y) \ne (0,0) \) then Theorem 2.3.1 (b) indicates that for any open rectangle that does not contain \( (0,0) \) then there exists a unique solution for (2.3.4)
Determine whether the IVP has a unique solution using Theorem 2.3.1 \begin {equation} \label {eq:2.3.5} \frac{dy}{dx} = \frac{x+y}{x-y} \end {equation}
Solution
Identify \( f(x,y) \) using (2.3.1) and check whether it is continuous: \begin{equation*} \frac{dy}{dx} = \overset{f(x,y)}{\overbrace{\frac{x+y}{x-y}}} \end{equation*} Then the denominator \( x-y = 0 \) then the function is undefined which is whever \( y = x.\) So, aside from the line \( y = x \) the function \( f(x,y) = \displaystyle \frac{x+y}{x-y} \) is continuous for all \((x,y) \) so Theorem 2.3.1 (a) indicates that there exists at least one solution to the IVP.
So we know there exists a solution, now we will find out if it is unique by finding the partial derivative with respect to \( y \) \begin{align*} f_y(x,y) = & \frac{\partial}{\partial y}\left(\frac{x+y}{x-y}\right) \\ = & \frac{\left(x-y\right) \frac{\partial}{\partial y} \left(x+y\right) - \left(x+y\right) \frac{\partial}{\partial y} \left(x-y\right)}{\left(x-y\right)^2} \\ = & \frac{\left(x-y\right)\left(1\right) - \left(x+y\right) \left(-1\right)}{\left(x-y\right)^2} \\ = & \frac{\left(x-y\right) + \left(x+y\right) }{\left(x-y\right)^2} \\ = & \frac{ 2x }{\left(x-y\right)^2} \\ \end{align*} Again aside from the line \( y = x, \) the derivative \( f_x(x,y) = \frac{ 2x }{\left(x-y\right)^2} \) is continuous for all other \((x,y). \) Since both \( f(x,y) \) and \( f_x(x,y) \) are continuous for all \((x,y) \) not on the line \( y = x \) then Theorem 2.3.1 (b) indicates that for any open rectangle that does not contain part of the line \( y = x \) then there exists a unique solution for (2.3.5)
In Example  2.2.4 we found that the solutions of \begin {equation} \label {eq:2.3.6} \frac{dy}{dx}=2xy^2 \end {equation} are \[ y\equiv 0\quad \mbox{ and } \quad y=-{1 \over x^2+c}\] where \(c\) is an arbitrary constant. In particular, this implies that no solution of (2.3.6) other than \(y\equiv 0\) can equal zero for any value of \(x\). Show that Theorem  2.3.1 (b) implies this.
Solution
We’ll obtain a contradiction by assuming that (2.3.6) has a solution \(y_1\) that equals zero for some value of \(x\), but isn’t identically zero. If \(y_1\) has this property, there’s a point \(x_0\) such that \(y_1(x_0)=0\), but \(y_1(x)\ne 0\) for some value of \(x\) in every open interval that contains \(x_0\). This means that the initial value problem \begin {equation} \label {eq:2.3.7} \frac{dy}{dx}=2xy^2,\quad y(x_0)=0 \end {equation} has two solutions \(y\equiv 0\) and \(y=y_1\) that differ for some value of \(x\) on every open interval that contains \(x_0\). This contradicts Theorem  2.3.1 (b), since in (2.3.6) the functions \[ f(x,y)=2xy^2 \quad \mbox{ and } \quad f_y(x,y)= 4xy\] are both continuous for all \((x,y)\), which implies that (2.3.7) has a unique solution on some open interval that contains \(x_0\).
Determine whether the IVP has a unique solution using Theorem 2.3.1 \begin {equation} \label {eq:2.3.8} \frac{dy}{dx}= \frac{10}{3}xy^{\frac{2}{5}} \quad\quad y(x_0)=y_0 \end {equation}
Solution
Identify \( f(x,y) \) using (2.3.1) and check whether it is continuous: \begin{equation*} \frac{dy}{dx} = \overset{f(x,y)}{\overbrace{\frac{10}{3}xy^{\frac{2}{5}}}} \end{equation*} Since the root is odd, it is defined for all \(y\) so \( f(x,y) = \displaystyle \frac{10}{3}xy^{\frac{2}{5}} \) is continuous for all \((x,y)\) so Theorem 2.3.1 (a) indicates that there exists at least one solution to the IVP.
So we know there exists a solution, now we will find out if it is unique by finding the partial derivative with respect to \( y \) \begin{align*} f_y(x,y) = & \frac{\partial}{\partial y}\left( \frac{10}{3}xy^{\frac{2}{5}} \right) \\ = & \frac{10}{3} \cdot \frac{2}{5} xy^{-\frac{3}{5}} \\ = & \frac{4x}{3y^{\frac{3}{5}}} \\ \end{align*} While the function \( f(x,y) = \displaystyle \frac{10}{3}xy^{\frac{2}{5}} \) is continuous for all \((x,y),\) the partial derivative \( f_y(x,y) = \frac{4x}{3y^{\frac{3}{5}}} \) is discontinuous when \( y = 0 \) so Theorem 2.3.1 (b) indicates that for any open rectangle that does not contain \( y = 0 \) then there exists a unique solution for (2.3.3)
Show that the IVP has more than one solution on every open interval that contains \(x_0=0\) \begin {equation} \label {eq:2.3.9} \frac{dy}{dx}={10 \over 3}xy^{\frac{2}{5}} \quad \quad y(0)=0 \end {equation}
Solution
In Example 2.3.5 we showed that the IVP (2.3.8) has a unique solution on any open rectangle that does not contain the \(x-\)axis \(( y = 0 )\) meaning that if the open rectangle does contain the \(x-\)axis then the IVP will have a solution, but it will be non-unique
To show that the IVP (2.3.9) has more than one solution, first notice it has a trivial solution \( y(x) \equiv 0 \) which we can verify by substitution: \begin{align} \frac{dy}{dx} = & {10 \over 3}xy^{\frac{2}{5}} \\ \frac{d}{dx} \Big( 0 \Big) = & {10 \over 3} x \Big(0\Big)^{\frac{2}{5}} \quad \checkmark \nonumber \end{align} Now we seek nontrivial solutions where \( y \ne 0 \) and divide by \( y^{\frac{2}{5}} \) to separate the variables \begin{align} \frac{dy}{dx} = & {10 \over 3}xy^{\frac{2}{5}} \nonumber \\ \frac{1}{y^{\frac{2}{5}}} \cdot \frac{dy}{dx} = & {10 \over 3}x \nonumber \\ \int \Big( \frac{1}{y^{\frac{2}{5}}}\frac{dy}{dx} \Big) dx = & \int \Big( {10 \over 3}x \Big) dx \nonumber \\ \int y^{-\frac{2}{5}} dy = & {10 \over 3} \int x dx \nonumber \\ \frac{5}{3} y^{\frac{3}{5}} = & {10 \over 3} \left(\frac{1}{2}x^2\right) + k \nonumber \\ \frac{5}{3} y^{\frac{3}{5}} = & {5 \over 3} x^2 + k \nonumber \\ y^{\frac{3}{5}} = & \frac{3}{5} \cdot {5 \over 3} x^2 + \underset{c}{\underbrace{ \frac{3}{5} k }} \nonumber \\ y^{\frac{3}{5}} = & x^2 + c \nonumber \\ y= & \left( x^2 + c \right)^{ \frac{5}{3} } \end{align} So we have found the one-parameter family of functions (2.3.11) that is a solution of the differential equation (2.3.10). Note that the one-parameter family (2.3.11) is defined for all \( x \) therefore has IOV \( (-\infty,\infty) \) and regardless of any choice of \( c \) the one-parameter family will not yield the trivial solution, so therefore the general solution of the differential equation (2.3.10) includes the one-parameter family and the trivial solution, which is a singular solution: \begin {equation*} y(x) = 0 \quad \quad y(x) = \left( x^2 + c \right)^{ \frac{5}{3} } \end {equation*}
Differentiate (2.3.11) with respect to \(x \) using the power rule and chain rule
Differentiating (2.3.11), we find: \begin{align*} \frac{dy}{dx}= & \frac{10}{3} x \left( x^2 + c \right)^{ \frac{2}{3} } \end{align*}
Substituting into (2.3.10) \begin{align*} \overset{\frac{dy}{dx}}{\overbrace{ \frac{10}{3} x \left( x^2 + c \right)^{ \frac{2}{3} } }} \quad \overset{?}{=} & \quad {10 \over 3} x \Bigg( \overset{y}{ \overbrace{ \left( x^2 + c \right)^{ \frac{5}{3} } } } \Bigg)^{\frac{2}{5}} \\ \frac{10}{3} x \left( x^2 + c \right)^{ \frac{2}{3} } = & {10 \over 3} x \left( x^2 + c \right)^{ \frac{2}{3} } \quad \checkmark \end{align*} So we have verified that (2.3.11) is a solution of (2.3.10) on \( (-\infty,\infty) \)
Now we apply the initial condition \( y(0) = 0 \) to solve the IVP (2.3.9) by finding \( c \) \begin{align*} \underset{0}{\underbrace{y(0)}} = & \left( 0^2 + c \right)^{ \frac{5}{3} } \\ 0= & c^{ \frac{5}{3} } \Longrightarrow c = 0 \\ \end{align*} Substituting in \( c = 0 \) to (2.3.11), we obtain a particular solution to the IVP (2.3.9) \begin{align*} y(x) = & x^{ \frac{10}{3} } \end{align*}
Now we have two solutions to the IVP, one is from the one-parameter family \( y(x) = x^{ \frac{10}{3} } \) and one is the trivial solution \( y(x) = 0 \) so while the IVP definitely has a solution, it is not unique. Both solutions are defined on \((-\infty ,\infty )\), and they differ on every open interval that contains \(x_0=0\) (see Figure 2.3.2.)
In fact, there are four distinct solutions of (2.3.9) defined on \((-\infty ,\infty )\) that differ from each other on every open interval that contains \(x_0=0\). Can you identify the other two?
From Example 2.3.5, the IVP \begin {equation} \label {eq:2.3.12} \frac{dy}{dx}={10 \over 3}xy^{2/5} \quad\quad y(0)=-1 \end {equation} has a unique solution on some open interval that contains \(x_0=0\). Find a solution and determine the largest open interval \((a,b)\) on which it’s unique.
Solution
From Example 2.3.6, we found the general solution to the differential equation: \begin {equation*} y(x) = 0 \quad \quad y(x) = \left( x^2 + c \right)^{ \frac{5}{3} } \end {equation*}
Now we apply the initial condition \( y(0) = -1 \) to solve the IVP (2.3.9) by finding \( c \) \begin{align*} \underset{-1}{\underbrace{y(0)}} = & \left( 0^2 + c \right)^{ \frac{5}{3} } \\ -1= & c^{ \frac{5}{3} } \Longrightarrow c = -1 \\ \end{align*} Substituting in \( c = -1 \) to (2.3.11), we obtain a particular solution to the IVP (2.3.12) \begin{align} \label {eq:2.3.13} y(x) = & \left(x^2 - 1 \right)^{ \frac{5}{3} } \end{align} Note that (2.3.13) is from the one-parameter family (2.3.11) and is a solution of (2.3.12) on IOV \((-\infty ,\infty )\) and \((-1,1)\) is the largest open interval on which (2.3.12) has a unique solution, shown in Figure 2.3.3.
From Exercise 2.2.15, there are infinitely many other solutions of (2.3.12) that differ from (2.3.13) on every open interval larger than \((-1,1)\). One such solution is \[ y = \left \{ \begin {array}{cl} (x^2-1)^{5/3} & -1 \le x \le 1 \\[6pt] 0 & |x|>1\end {array} \right. \]
From Example 2.3.5, the initial value problem \begin {equation} \label {eq:2.3.14} \frac{dy}{dx}={10 \over 3}xy^{2/5} \quad \quad y(0)=1 \end {equation} has a unique solution on some open interval that contains \(x_0=0.\) Find the solution and determine the largest open interval on which it’s unique.
Solution
From Example 2.3.6, we found the general solution to the differential equation: \begin {equation*} y(x) = 0 \quad \quad y(x) = \left( x^2 + c \right)^{ \frac{5}{3} } \end {equation*}
Now we apply the initial condition \( y(0) = 1 \) to solve the IVP (2.3.14) by finding \( c \) \begin{align*} \underset{1}{\underbrace{y(0)}} = & \left( 0^2 + c \right)^{ \frac{5}{3} } \\ 1= & c^{ \frac{5}{3} } \Longrightarrow c = 1 \\ \end{align*} Substituting in \( c = 1 \) to (2.3.11), we obtain a particular solution to the IVP (2.3.12) \begin{align} \label {eq:2.3.15} y(x) = & \left(x^2 + 1 \right)^{ \frac{5}{3} } \end{align} Note that (2.3.15) is from the one-parameter family (2.3.11) and is a solution of (2.3.14) on IOV \((-\infty ,\infty )\)
