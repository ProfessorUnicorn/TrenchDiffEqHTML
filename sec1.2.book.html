1.2  BASIC CONCEPTS
Basic Concepts
A differential equation is an equation that contains one or more derivatives of an unknown function. The order of a differential equation is the order of the highest derivative that it contains. A differential equation is an ordinary differential equation if it involves an unknown function of only one variable, or a partial differential equation if it involves partial derivatives of a function of more than one variable. For now we’ll consider only ordinary differential equations, and we’ll just call them differential equations.
Throughout this text, all variables and constants are real unless stated otherwise. We’ll usually use \(x\) for the independent variable unless the independent variable is time; then we’ll use \(t\).
The simplest differential equations are first order equations of the form \[ {dy\over dx}=f(x) \] where \(f=f(x)\) is a known function of \(x\).
When a linear differential equation only has the derivative of the dependent variable present and not the dependent variable itself \( \big( \)i.e. when \( p(x)=0 \)\( \big) \), we will call the equation integrable since we can immediately find the antiderivative using integration. \begin {align*} \frac{dy}{dx} = & f(x) \Longrightarrow y = c + \int f(x) dx \end {align*}
We already know from calculus from the power rule how to find functions that satisfy this kind of equation by finding the antiderivative. For example, if \[ \frac{dy}{dx}=x^3\] then \[ y=\int x^3 dx=\frac{1}{4} x^4 + c \] where \(c\) is an arbitrary constant.
We can even solve second order differential equations of this form \[ {d^2y\over dx^2}=x^3 \] Again we solve by finding the antiderivative: \[ \frac{dy}{dx} = \int \Bigg({d^2y\over dx^2} \Bigg) dx = \int x^3 dx = \frac{1}{4} x^4 + c \] But this time we simply find the antiderivative twice: \[ y = \int \Bigg( \frac{dy}{dx} \Bigg) dx = \int \Big( \frac{1}{4} x^4 + c_1 \Big) dx = \frac{1}{20} x^5 + c_1x + c_2 \]
Use integral properties to show \( \displaystyle \int \Big( \frac{1}{4} x^4 + c_1 \Big) dx = \frac{1}{20} x^5 + c_1x + c_2 \)
Again we can solve any other simple higher order differential equations by repeated antidifferentiation from calculus. \begin {equation} \label {eq:1.2.1} \frac{d^ny}{dx^n}=f(x) \end {equation}
Solve the differential equation \( \displaystyle \frac{dy}{dx} = \sqrt[5]{x} \)
Solve the differential equation \( \displaystyle \frac{d^2y}{dx^2} = \frac{1}{x^8} \)
Solve the differential equation \( \displaystyle \frac{d^3y}{dx^3} = \cos x \)
Solve the differential equation \( \displaystyle \frac{d^4y}{dx^4} = e^x \)
Except for illustrative purposes in this section, there’s no need to consider differential equations like (1.2.1) and we’ll usually consider differential equations that can be written as \begin {equation} \label {eq:1.2.2} \frac{d^ny}{dx^n}=f\left(x,y,\frac{dy}{dx}, \dots ,\frac{d^{n-1}y}{dx^{n-1}}\right) \end {equation} where either the function itself, \(y\), or one of its derivatives \(\displaystyle \frac{dy}{dx}\), …, \(\displaystyle \frac{d^{n-1}y}{dx^{n-1}}\) actually appears on the right. Here are some examples: \[ \begin {array}{rcll} \displaystyle {dy\over dx}-x^2&=&0&\mbox { (first order)}\\ \\ \displaystyle {dy\over dx}+2xy^2&=&-2&\mbox { (first order)}\\ \\ \displaystyle {d^2y\over dx^2}+2\displaystyle {dy\over dx}+y&=&2x&\mbox { (second order)}\\ \\ \displaystyle x\frac{d^3y}{dx^3}+y^2&=&\sin x &\mbox { (third order)}\\ \\ \displaystyle \frac{d^ny}{dx^n}+x\frac{dy}{dx}+3y&=&x&\mbox { ($n$-th order)} \end {array} \] Although none of these equations is written as in (1.2.2), all of them can be written in this form: \[ \begin {array}{rcl} \displaystyle \frac{dy}{dx}&=&x^2\\ \\ \displaystyle \frac{dy}{dx}&=&-2-2xy^2\\ \\ \displaystyle \frac{d^2y}{dx^2}&=&2x-2\frac{dy}{dx}-y\\ \\ \displaystyle \frac{d^3y}{dx^3}&=&\displaystyle {\sin x-y^2\over x}\\ \\ \displaystyle \frac{d^ny}{dx^n}&=&x - x \displaystyle \frac{dy}{dx}-3y \end {array} \]
For each of these equations, identify the right side function and write in function notation. For instance, \( \displaystyle \frac{dy}{dx} = x^2 \) has the right side function \( f(x) = x^2 \)
Solutions of Differential Equations
A solution of a differential equation is a function that satisfies the differential equation on some open interval; thus, \(y\) is a solution of (1.2.2) if \(y\) is \(n\) times differentiable and \[ \frac{d^ny}{dx^n}=f\left(x,y,\frac{dy}{dx}, \dots ,\frac{d^{n-1}y}{dx^{n-1}}\right) \] for all \(x\) in some open interval \((a,b)\). In this case, we also say that \(y\) is a solution of (1.2.2) on \((a,b)\). Functions that satisfy a differential equation at isolated points are not interesting. For example, \(y=x^2\) satisfies \[ x\frac{dy}{dx}+x^2=3x \] if and only if \(x=0\) or \(x=1\), but it’s not a solution of this differential equation because it does not satisfy the equation on an open interval.
Show that \( y = x^2 \) satisfies the \( x\frac{dy}{dx}+x^2=3x \) if and only if \(x=0\) or \(x=1\)
The graph of a solution of a differential equation is a solution curve. More generally, a curve \(C\) is said to be an integral curve of a differential equation if every function \(y=y(x)\) whose graph is a segment of \(C\) is a solution of the differential equation. Thus, any solution curve of a differential equation is an integral curve, but an integral curve need not be a solution curve.
The circle \begin {equation} \label {eq:1.2.3} x^2+y^2=9 \end {equation} is an integral curve of the differential equation \begin {equation} \label {eq:1.2.4} \frac{dy}{dx}=-{x\over y} \end {equation}
Although the circle (1.2.3) is not a function, we can solve for \( y \) to obtain two separate functions that are segments of the circle: \[ y_1=\sqrt {9-x^2}\quad \mbox{ and } \quad y_2=-\sqrt {9-x^2}\]
Solve \( x^2+y^2=9 \) explicitly for \( y \) and show you get the semi-circle segments \( y_1 \) and \( y_2 \)
By differentiating and substituting, it can be easily verified that both of the semi-circle curves, \( y_1 \) and \( y_2 \), are solutions to the differential equation.
Show that \( y_1 \) and \( y_2 \) satisfies \( \displaystyle \frac{dy}{dx}=-{x\over y}\) on the open interval \((-3, 3)\)
Note that all the segments of the circle that are functions (i.e. the semi-circles \( y_1 \) and \( y_2 \) ) are solutions to the differential equation (1.2.4) and thus called solution curves. Since all of the segments of the circle are solutions, the circle itself is an integral curve of the equation, but it is not a solution curve since the circle itself is not the graph of a function and is not itself a solution.
Verify that (1.2.5) is a solution of (1.2.6) on \((0,\infty )\) and on \((-\infty ,0)\). \begin {equation} \label {eq:1.2.5} y={x^2\over 3}+{1\over x} \end {equation} \begin {equation}\label {eq:1.2.6} x\frac{dy}{dx}+y=x^2 \end {equation}
Calculate \( \displaystyle \frac{dy}{dx} \) for \( \displaystyle y={x^2\over 3}+{1\over x} \) using the power rule
Solution
Substituting \( y \) (1.2.5) and \( \displaystyle \frac{dy}{dx}={2x\over 3} - {1\over x^2} \) into (1.2.6) yields \[ x\left.\frac{dy}{dx}\right|_x+y(x)=x \left ({2x\over 3} - {1\over x^2}\right )+ \left ({x^2\over 3}+{1\over x}\right )=x^2 \] Note that \( y \) is undefined at \( x = 0 \) so therefore \(y\) is a solution of (1.2.6) on \((-\infty ,0)\) and \((0,\infty )\). However, \(y\) isn’t a solution of the differential equation on any open interval that contains \(x=0\), since \(y\) is not defined at \(x=0\).
Figure 1.2.1 shows the graph of the integral curve (1.2.5). The segment of the graph of (1.2.5) on \((0,\infty )\) is a solution curve of (1.2.6), as is the segment of the graph on \((-\infty ,0)\).
Show that if \(c_1\) and \(c_2\) are constants then (1.2.7) is a solution of (1.2.8) on \((-\infty ,\infty )\) \begin {equation} \label {eq:1.2.7} y = 2x - 4 + (c_1+c_2x) e^{-x} \end {equation} \begin {equation} \label {eq:1.2.8} \frac{d^2y}{dx^2} + 2 \frac{dy}{dx} + y = 2x \end {equation}
Calculate \( \displaystyle \frac{dy}{dx} \) and \( \displaystyle \frac{d^2y}{dx^2} \) for \( \displaystyle y=(c_1+c_2x)e^{-x}+2x-4 \) using the product rule
Solution
Differentiating (1.2.7) twice yields \[ \frac{dy}{dx} = 2 + (-c_1 + c_2 -c_2x) e^{-x} \quad \text{and} \quad \frac{d^2y}{dx^2} = (c_1 - 2c_2 + c_2x) e^{-x} \] Substituting in to (1.2.8) \begin{align*} \begin{array}{rlllllrll} \displaystyle \frac{d^2y}{dx^2} = & & & & & ( & c_1 & - & 2c_2 & + & c_2 x & )e^{-x} \\ 2 \displaystyle \frac{dy}{dx} = & & & 4 & + & ( & -2c_1 & + & 2c_2 & - & 2c_2 x & )e^{-x} \\ + \quad \quad \quad \quad \quad y = & 2x & - & 4 & + & ( & c_1 & & & + & c_2 x & )e^{-x} \\ \hline \\ \displaystyle \frac{d^2y}{dx^2} + 2 \displaystyle\frac{dy}{dx} + y = & 2x & & & + & ( & \overset{1-2+1}{\overbrace{0}}c_1 & + & \overset{-2+2}{\overbrace{0}}c_2 & + & \overset{1-2+1}{\overbrace{0}} c_2x & )e^{-x} \\ \\ \displaystyle \frac{d^2y}{dx^2} + 2 \displaystyle\frac{dy}{dx} + y \overset{\checkmark}{=} & 2x & & & & & & & & & & \\ \end{array} \end{align*} \[ \] for all values of \(x\). Therefore (1.2.7) is a solution of (1.2.8) on \((-\infty ,\infty )\)
Find all solutions of \begin {equation} \label {eq:1.2.9} \frac{d^ny}{dx^n}=e^{2x}\end {equation}
Solution
Solution   Antidifferentiating repeatedly (1.2.9) yields \begin{align*} \frac{d^{n-1}y}{dx^{n-1}} = & \frac{1}{2} e^{2x} + k_1 \\ \\ \frac{d^{n-2}y}{dx^{n-2}} = & \frac{1}{4} e^{2x} + k_1x + k_2 \\ \\ \frac{d^{n-3}y}{dx^{n-3}} = & \frac{1}{8} e^{2x} + \frac{1}{2}k_1x^2 + k_2x + k_3 \\ \\ \frac{d^{n-4}y}{dx^{n-4}} = & \frac{1}{16} e^{2x} + \frac{1}{6}k_1x^3 + \frac{1}{2}k_2x^2 + k_3x + k_4 \\ \\ \frac{d^{n-5}y}{dx^{n-5}} = & \frac{1}{32} e^{2x} + \frac{1}{24}k_1x^4 + \frac{1}{6}k_2x^3 + \frac{1}{2}k_3x^2 + k_4x + k_5 \\ \\ \end{align*}
Show all the work for calculating the first five antiderivatives
Notice the pattern on the \( e^{2x} \) terms; the \( n^{th} \) derivative has a factor of \( \displaystyle \frac{1}{2^n} \): \[ \frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \frac{1}{32}, \dots \]
Notice the pattern on the rest of the terms; the coefficients are factorials: \[ \underset{0!}{\underbrace{1}}, \underset{1!}{\underbrace{1}}, \underset{2!}{\underbrace{2}}, \underset{3!}{\underbrace{6}}, \underset{4!}{\underbrace{24}}, \dots \]
Incorporating these patterns, flipping the order of the polynomial terms, and renaming the arbitrary constants, we can rewrite the fifth antiderivative more simply: \begin{align*} \frac{d^{n-5}y}{dx^{n-5}} = & \frac{1}{32} e^{2x} + \frac{1}{24}k_1x^4 + \frac{1}{6}k_2x^3 + \frac{1}{2}k_3x^2 + k_4x + k_5 \\ = & \frac{1}{2^5} e^{2x} + \frac{1}{4!}k_1x^4 + \frac{1}{3!}k_2x^3 + \frac{1}{2!}k_3x^2 + \frac{1}{1!}k_4x + \frac{1}{0!}k_5 \\ = & \frac{1}{2^5} e^{2x} + \frac{1}{0!}K_0 + \frac{1}{1!} K_1 x + \frac{1}{2!} K_2 x^2 + \frac{1}{3!} K_3 x^3 + \frac{1}{4!} K_4 x^4 \\ = & \frac{1}{2^5} e^{2x} + \underbrace{ c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 } \\ = & \frac{1}{2^5} e^{2x} + \sum_{n=0}^{4} { c_n x^n } \end{align*}
Therefore all solutions of (1.2.9) can be written as \begin {equation} \label {eq:1.2.10} y = \frac{1}{2^n} e^{2x} + c_0 + c_1 x + c_2 x^2 + \cdots +c_n x^n = \frac{1}{2^n} e^{2x} + \sum_{n=0}^{n} { c_n x^n } \end {equation} where we renamed the arbitrary constants to make the formula as simple as possible. As a general rule, arbitrary constants appearing in solutions of differential equations should be simplified whenever possible.

Initial Value Problems
In Example 1.2.4 we found that the differential equation \( \displaystyle \frac{d^ny}{dx^n}=e^{2x}\) has an infinite family of solutions that depend upon the \(n\) arbitrary constants \(c_1\), \(c_2\), …, \(c_n\). In the absence of additional conditions, there’s no reason to prefer one solution of a differential equation over another. However, we’ll often be interested in finding a particular solution of a differential equation that satisfies one or more specific initial conditions which we call an initial value problem.
Find a solution of the differential equation such that \(y(1)=2\) \[ \frac{dy}{dx}=x^3 \]
Solution
At the beginning of this section we found the general solution of this equation: \[ y = \frac{1}{4} x^4 + c \] We can determine a value of \(c\) that satisfies the initial condition \(y(1)=2\), by simply evaluating the solution at \( x=2 \): \begin{align*} y(x) = & \frac{1}{4}x^{4} + c \\ \\ \underset{2}{\underbrace{y(1)}} = & \frac{1}{4}\left(1\right)^{4} + c \\ 2 = & \frac{1}{4}+ c \Longrightarrow c = \frac{7}{4} \\ \end{align*} Substituting in \( c = \frac{7}{4} \), we obtain the particular solution: \[ y(x) = \frac{1}{4}x^{4} + \frac{7}{4} = \frac{1}{4}\left(x^2 + 7\right) \]
Figure 1.2.2 shows the graph of this solution. Note that imposing the condition \(y(1)=2\) is equivalent to requiring the graph of \(y\) to pass through the point \((1,2)\).
We can rewrite the problem considered in Example 1.2.5 more briefly as \[ \frac{dy}{dx}=x^3 \quad y(1)=2\] We call this an initial value problem and the requirement \(y(1)=2\) is an initial condition. Initial value problems can also be posed for higher order differential equations. For instance, \begin {equation} \label {eq:1.2.11} \frac{d^2y}{dx^2} - 2\frac{dy}{dx}+3y=e^x \quad y(0)=1 \quad \left.\frac{dy}{dx}\right|_0=2 \end {equation} is an initial value problem for a second order differential equation where \(y\) and \(\displaystyle \frac{dy}{dx}\) are required to have specified values at \(x=0\). In general, an initial value problem for an \(n^{th}\) order differential equation requires \(n \) initial conditions specified on \(y\) and its first \(n-1\) derivatives at some initial point \(x_0\).
We’ll denote an initial value problem for a differential equation by writing the initial conditions after the differential equation, as in (1.2.11). For example, we would write an initial value problem for (1.2.2) as \begin {equation}\label {eq:1.2.12} \frac{d^ny}{dx^n}=f\left(x,y,\frac{dy}{dx}, \dots ,\frac{d^{n-1}y}{dx^{n-1}}\right) \quad y(x_0)=k_0 \quad \left.\frac{dy}{dx}\right|_{x_0}=k_1 \quad \dots \quad \frac{d^{n-1}y}{dx^{n-1}}=k_{n-1} \end {equation} Consistent with our earlier definition of a solution of the differential equation in (1.2.12), we say that \(y\) is a solution of the initial value problem (1.2.12) if \(y\) is \(n\) times differentiable and \[ \frac{d^ny}{dx^n}=f\left(x,y,\frac{dy}{dx}, \dots ,\frac{d^{n-1}y}{dx^{n-1}}\right) \] for all \(x\) in some open interval \((a,b)\) that contains \(x_0\), and \(y\) satisfies the initial conditions in (1.2.12). The largest open interval that contains \(x_0\) on which \(y\) is defined and satisfies the differential equation is called the interval of validity of \(y\).
In Example 1.2.5 we found that \begin {equation} \label {eq:1.2.13} y = \frac{1}{4} \left( x^4 + 7 \right) \end {equation} is a solution of the initial value problem \[ \frac{dy}{dx} = x^3 \quad y(1) = 2 \] Since the function in (1.2.13) is defined for all \(x\), the interval of validity of this solution is \((-\infty ,\infty ) \)
\begin {equation} \label{eq:1.2.14} y={x^2\over 3}+{1\over x} \end {equation} In Example 1.2.2 we verified that (1.2.14), is the solution of the following differential equation with intervals of validity \((0,\infty )\) and \((-\infty ,0)\): \[ x \frac{dy}{dx} + y = x^2 \] By evaluating (1.2.14) at \(x= 1\) and \(x= -1\), we can show that it is a solution of the following initial value problems: \begin{equation} \label {eq:1.2.15} x\frac{dy}{dx}+y=x^2 \quad y(1)={4\over 3} \end {equation} and \begin{equation} \label {eq:1.2.16} x \frac{dy}{dx} + y = x^2 \quad y(-1) = -{2\over 3} \end {equation}
Show that (1.2.14) is a solution of the initial value problems (1.2.15) and (1.2.16)
The interval of validity of (1.2.14) as a solution of (1.2.15) is \((0,\infty )\), since this is the largest interval that contains \(x=1\) on which (1.2.14) is defined. Similarly, the interval of validity of (1.2.14) as a solution of (1.2.16) is \((-\infty ,0)\), since this is the largest interval that contains \(x=-1\) on which (1.2.14) is defined.
Free Fall Under Constant Gravity
The term initial value problem originated in problems of motion where the independent variable is \(t\) (representing elapsed time), and the initial conditions are the position and velocity of an object at the initial time of an experiment.
An object falls under the influence of gravity near Earth’s surface, where it can be assumed that the magnitude of the acceleration due to gravity is a constant \(g\). Using Newton’s 2nd law of motion, construct a mathematical model for the motion of the object in the form of an initial value problem for a second order differential equation, assuming that the height and velocity of the object at time \(t=0\) are known. Assume that gravity is the only force acting on the object. Solve the initial value problem to obtain the height as a function of time
Solution
Recall Newton’s 2nd law of motion, \( F = ma \), the force \( F \) acting on the object is equal to the mass \( m \) of the object times the acceleration \( a \) of the object. Acceleration is the derivative of velocity and velocity is the derivative of position so position is the second derivative of acceleration and can be written as \( a = \ddot{y} \) where \(y(t)\) represent the position (height) of the object at time \(t\). The only force acting on the object is gravity and the acceleration due to gravity is in the downward (negative) direction, so the total force \( F \) is given by \( -mg \). Thus Newton’s 2nd law of motion can be rewritten to represent a second order differential equation modeling this situation: \begin{equation*} F = ma \Longrightarrow -mg = m \ddot{y} \Longrightarrow -g = \ddot{y} \end{equation*} where the dot notation indicates differentiation with respect to \(t\). If \(y_0 \) denotes the initial height and \(v_0\) denotes the initial velocity when \(t=0\), then \(y\) is a solution of the initial value problem \begin {equation} \label {eq:1.2.17} \ddot{y}=-g \quad y(0)=y_0 \quad \dot{y}(0)=v_0 \end {equation} Which can be solved by antidifferentiating twice: \begin {align*} \ddot{y} = & -g \\ \Longrightarrow \dot{y} = & -gt + c_1 \\ \Longrightarrow y = & -\frac{1}{2}gt^2 + c_1 t + c_2 \\ \end {align*}
Show all the work for calculating the two antiderivatives
Imposing the initial conditions \(y(0)=y_0\) and \( \dot{y}(0)=v_0 \) in these two equations shows that \(c_1=v_0\) and \(c_2=y_0\). Therefore the solution of the initial value problem (1.2.17) is \[ y = - \frac{1}{2} gt^2 + v_0 t + y_0 \]
